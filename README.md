# A Gentle Introduction to Python for Research Computing

1. Clone this repository
2. Set up your python environment according to [The Carpentries](https://carpentries.github.io/workshop-template/install_instructions/#python-1)
3. Run the setup script `./setup.sh`
4. Either follow the Carpentries guide: [Programming with Python](https://swcarpentry.github.io/python-novice-inflammation/) or the included `pres.pdf`.
